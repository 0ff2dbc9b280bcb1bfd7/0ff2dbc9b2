Open New Terminal Tab Using Command in Linux

Oct 29th, 2016 10:08 am

Linux default (gnome in Ubuntu) terminal doesn’t provide an API / Command to open a new tab, however new window can be opened using gnome-terminal command. Terminal provides a shortcut Ctrl+Shift+t to open a new tab, leveraging this we can use xdotool to emulate the new tab shortcut.

xdotool according to Linux.org:

The xdotool is a utility used from the terminal or in a script to manually perform keyboard input. The commands can also be used to make a script of many xdotool commands to perform large tasks. Later articles will cover the xdotool ability to perform mouse input as well as window and desktop manipulation.

Install xdotool:

# Ubuntu

sudo apt-get install xdotool

# Fedora / CentOS:

sudo yum install xdotool

Now trigger the new tab shortcut using: xdotool key ctrl+shift+t

Create a alias for ease: alias newtab=”xdotool key ctrl+shift+t”
