Convert Ogv to Images in Ubuntu

Dec 10th, 2016 12:34 pm

Convert ogv files to images using mplayer in two simple steps:

Step 1 – Install mplayer:

sudo apt-get install mplayer

Step 2 – Convert:

mplayer -ao null out.ogv -vo jpeg:outdir=output

This will create jpeg images file in a directory names “output” as specified.
