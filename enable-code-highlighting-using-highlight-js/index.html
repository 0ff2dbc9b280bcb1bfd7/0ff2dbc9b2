Enable Code Highlighting Using HighlightJS

Oct 30th, 2016 10:46 am

HighlightJS is the simplest way to enable syntax highlighting on the web, I’v tried Google code prettify, but found it a little more complex than highlight js. You will be able to get highlight js up and running in 4 simple steps:

Step 1: Download:

Go to highlightjs.org/download to download the library, I recommend to customize the download by selecting the languages you use, instead of the whole library.

Step 2: Import:

Once you download the library (I’m using a customized version), you should be having two important things, 1 the highlight.pack.js file and 2 the styles folder.Â Styles folder have multiple styles (themes) that you can choose from, I’m using github-gist style, which is very similar to the gist.github.com. Now import style and js into html:

<link rel="stylesheet" type="text/css" href="github-gist.css">
<script src="highlight.pack.js"></script>

Step 3: Add code:

Let’s add some code to the web page so that highlight js could highlight it, consider the below example:

<pre><code>function sayHello(){
    console.log("Hello!");
}
sayHello();</code></pre>

Step 4: Init HighlightJS:

Now, call the highlight JS init function by running:

hljs.initHighlightingOnLoad();
