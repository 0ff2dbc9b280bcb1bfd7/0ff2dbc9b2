Get Size of Directory in Linux

Dec 10th, 2016 1:43 pm

The command du “summarizes disk usage of each FILE, recursively for directories,” e.g. usage,

du -hs /path/to/directory
  • -h is to get the numbers “human readable”, e.g. get 140M instead of 143260 (size in KBytes)
  • -s is for summary (otherwise you’ll get not only the size of the folder but also for everything in the folder separately)

How to Install Debian .deb File in Ubuntu

Dec 10th, 2016 1:40 pm

Debian (.deb) packages are the packages that are used in Ubuntu. You can install any .deb package in your system. .deb files can generally be installed from your file manager (Nautilus) merely by clicking on them, since file associations with the default installer is already set in Ubuntu. These instructions are for those who wish to install packages from the command-line terminal (Terminal).

To install a downloaded Debian package (.deb): Open Terminal and type

sudo dpkg -i packagename.deb

To remove a Debian (Ubuntu) package (.deb):

sudo dpkg -r packagename

To Reconfigure/Repair an installed Debian (Ubuntu) package (.deb):

sudo dpkg-reconfigure packagename

Remove PPAs From Ubuntu or Any Debian Distro

Dec 10th, 2016 1:26 pm

Use the --remove flag, similar to how the PPA was added:

sudo add-apt-repository --remove ppa:whatever/ppa

As a safer alternative, you can install ppa-purge:

sudo apt-get install ppa-purge

And then remove the PPA, downgrading gracefully packages it provided to packages provided by official repositories:

sudo ppa-purge ppa_name

Note that this will uninstall packages provided by the PPA, but not those provided by the official repositories. If you want to remove them, you should tell it to apt:

sudo apt-get purge package_name

You can also remove PPAs by deleting the .list files from /etc/apt/sources.list.d directory.

Last but not least, you can also disable or remove PPAs from the “Software Sources” section in Ubuntu Settings with a few clicks of your mouse.

Convert Ogv to Gif in Ubuntu

Dec 10th, 2016 12:40 pm

Ogv files cannot be converted into gif right a way, we should convert ogv to images and then images to gif:

Convert ogv files to images usingÂ mplayer:

Step 1 – Install mplayer:

sudo apt-get install mplayer

Step 2 – Convert:

mplayer -ao null out.ogv -vo jpeg:outdir=output

This will create jpeg images file in a directory names “output” as specified.

Now convert images to gif:

We can use the pre installed “convert” tool to do this, run:

convert output/* output.gif

This will create a gif file “output.gif” with all the images inside output folder.

Tip – If you see that output.gif is too large, use the below command to optimise the gif:

convert output.gif -fuzz 10% -layers Optimize optimised.gif
