How Emit a Event When Ng-repeat Is Completed AngularJS

Dec 12th, 2016 8:03 pm

Use a directive with ng-repeat which triggers a function when ng-repeat reaches to last element, directive code:

.directive('onFinishEmit', function($timeout) {
    return {
        restrict: 'A',       
        link: function(scope, element, attr) {
            if (scope.$last === true) {
                $timeout(function() {                        
                    scope.$emit(attr.onFinishEmit);
                });
            }
        }
    }
});

Usage:

<div ng-repeat="data in dataset" on-finish-emit="renderComplete">

</div>

How to Read Process Output in Powershell

Dec 12th, 2016 7:43 pm

If you are running a process via powershell you can capture the stdout and stderr using StandardOutput.ReadToEnd() and StandardError.ReadToEnd(), below is the example:

$pinfo = New-Object System.Diagnostics.ProcessStartInfo
$pinfo.FileName = "ping.exe"
$pinfo.RedirectStandardError = $true
$pinfo.RedirectStandardOutput = $true
$pinfo.UseShellExecute = $false
$pinfo.Arguments = "localhost"
$p = New-Object System.Diagnostics.Process
$p.StartInfo = $pinfo
$p.Start() | Out-Null
$p.WaitForExit()
$stdout = $p.StandardOutput.ReadToEnd()
$stderr = $p.StandardError.ReadToEnd()
Write-Host "stdout: $stdout"
Write-Host "stderr: $stderr"
Write-Host "exit code: " + $p.ExitCode

Convert CR2 to JPG in Ubuntu

Dec 12th, 2016 7:41 pm

Convert cr2 to jpeg files in Ubuntu in a few simple steps:

Step 1: Install ufraw

sudo apt-get install ufraw

Step 2: For ufraw

for i in *.CR2; do ufraw-batch $i --out-type=jpeg --output $i.jpg; done;

This will convert all cr2 files a directory into jpeg files, later if you don’t want cr2 files use rm -f *.cr2
