Auto Update Chrome in Ubuntu

Oct 29th, 2016 11:06 am

Chrome does not provide any PPA to get auto updates, though there are a few 3rd aparty PPA, however they cannot be trusted, instead of using PPA anyways we can make use of a few simple set of commands to update chrome without much hassle, thanks to Chrome team for providing download to the latest version of Chrome with a simple URL, below are the steps:

  1. Remove Chrome: sudo dpkg –remove google-chrome-unstable
  2. Get latest version: **wget https://dl.google.com/linux/direct/google-chrome-beta_current_amd64.deb**
  3. Install: sudo dpkg -i google-chrome-beta_current_amd64.deb

TIP: Make use of a few additional commands to make this process even more smother:

  1. Kill Chrome: kill -9 $(pgrep -lf chrome | awk ‘{print $1}’)
  2. Remove previous Chrome .deb file: rm google-chrome-beta_current_amd64.deb

Connect to a Proxy in Google Chrome

Oct 29th, 2016 11:03 am

Chrome provides a command line flag –proxy-server which is used to connect to a proxy. Working example:

# This will connect to socks5 proxy.

google-chrome –proxy-server=”socks5://127.0.0.1:1080″

TIP: –proxy-server will bind a proxy to the current user session, i.e; if there is already a Chrome session running without a proxy and when you create another session using the above command with a proxy the proxy if not enforced. Use below command to create a new session with proxy server:

google-chrome –proxy-server=”socks5://127.0.0.1:1080″ –user-data-dir=/tmp/IWu9ddFgS6vynN –incognito –no-first-run http://ipinfo.io

Unlike opening Chrome in incognito mode via command line there is no straight forward flag to do the same with guest mode. However –user-data-dir flag can be used to leverage the guest mode like functionality.

Google Chrome uses a directory to store data specific to a given user, examples of this type of data are: history, bookmarks, and cookies. Hence we can use a temporary directory to create a Chrome instance and temporary directory will be deleted when Chrome session ends, this is exactly what we are looking for.

TIPS:

  • Use: –no-first-run to disable the popup asking to make Chrome default browser
  • Use: &> /dev/null at the end to will launch chrome without attaching to current tty.

Create a perfect alias:

Search for File Name Linux

Oct 29th, 2016 10:56 am

When searching for files in Linux, the often command that comes to mind is: ls | grep -i myfile, but ls is not capable of providing nested paths. find command along with grep can be used to recursively search file names and print absolute paths of the files matched:

find . -print | grep -i ‘myfile’

. represents current directory, feel free to change it according to your use, to search entire filesystem use /
