Disable Guest Account in Ubuntu

Dec 10th, 2016 12:30 pm

Guest accounts are not suitable at-times, and if you want to disable it, below the below 2 simple steps:

Step 1 – Edit config:

sudo gedit /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf

Step 2 – Add to config:

Add below like to config:

allow-guest=false

From next time you won’t be able to see guest account in the login screen, however if you would like to immediately test this out, do so by restarting – sudo restart

Create a Big Dummy File in Linux

Dec 10th, 2016 12:21 pm

In linux systems if you would like to create a huge size dummy file, make use of a pre installed tool fallocate. According to mandb:

fallocate is used to preallocate blocks to a file. For filesystems which support the fallocate system call, this is done quickly by allocating blocks and marking them as uninitialized, requiring no IO to the data blocks. This is much faster than creating a file by filling it with zeros.

Command:

fallocate -l 100M dummy.data

-l, –length length

Specifies the length of the allocation, in bytes.

This will create a file dummy.data with 100M, you can change 100M to any size are per your needs.

Parse CSV Into JSON in Node.JS

Nov 8th, 2016 6:24 am

csvtojson is a perfect library to convert CSV to JSON, if support’s api & as a command line tool.

Consider a sample csv as data.csv:

TIMESTAMP,UPDATE,UID,BYTES SENT,BYTES RCVED
1395426422,n,10028,1213,5461
1395426422,n,10013,9954,13560

Install csvtojson:

npm install csvtojson

Using csvtojson in Node.JS:

The below code will convert csv to json:

//Converter Class 
var Converter = require("csvtojson").Converter;
var converter = new Converter({});

//end_parsed will be emitted once parsing finished 
converter.on("end_parsed", function(jsonArray) {
    console.log(jsonArray); //here is your result jsonarray 
});

//read from file 
require("fs").createReadStream("./data.csv").pipe(converter);

When you run the above code, you get:

[{
    TIMESTAMP: 1395426422,
    UPDATE: 'n',
    UID: 10028,
    'BYTES SENT': 1213,
    'BYTES RCVED': 5461
}, {
    TIMESTAMP: 1395426422,
    UPDATE: 'n',
    UID: 10013,
    'BYTES SENT': 9954,
    'BYTES RCVED': 13560
}]

Using csvtojson as command line:

  1. Using as command: csvtojson ./myCSVFile

  2. Pipe the output: cat myCSVFile | csvtojson

Easy Way to Convert GIF File to MP4

Nov 7th, 2016 7:38 am

Gif’s files can be seamlessly converted to mp4 files using ffmpeg, using the below command:

ffmpeg -f gif -i file.gif file.mp4

 

If you don’t have ffmpeg installed, visit ffmpeg downloads page to download your os version, for Ubuntu you may follow the below steps:

sudo add-apt-repository ppa:mc3man/trusty-media
sudo apt-get update
sudo apt-get install ffmpeg
