Merge Multiple PDF Pages Into One Page in Linux

Oct 29th, 2016 12:46 pm

Merge two or more PDF pages into a single page, install pdfjam in Ubuntu use sudo apt-get install texlive-extra-utils, then run,

pdfjam input.pdf –nup 2×1 –landscape –outfile output.pdf

  • input.pdf: File to be merged
  • --nup 2x1: This denotes to merge 2 pages into a single page, to merge 4 pages into a single page use 4x1
  • --landscape: This will merge pages into landscape mode, this is recommended but optional
  • output.pdf: Output file (Generated file after the merge)

Extract Selected Pages From PDF in Linux

Oct 29th, 2016 12:46 pm

pdfjam is a handy tool to extract selected pages from a PDF. Install pdfjam, in Ubuntu use sudo apt-get install texlive-extra-utils, run:

pdfjam -o

An example of page ranges would be: {},3,67-70,80 ({} for insert an additional empty page)

Example:

pdfjam input.pdf {},101,103-106,{} -o output.pdf

Exclude a Folder in Linux While Using Zip Command

Oct 29th, 2016 12:10 pm

While compressing you might want to exclude a certain folder while including everything else, below is a example how to exclude .git folder and zip everything else, you can remane .gitwith any folder or filename.

zip -9 -r –exclude=’*.git*’ lg.zip ./

Multiple files / folders can be excluded using multiple --exclude flags, for example if you want to exclude .git and .git-backup use:

zip -9 -r –exclude=’*.git*’ –exlude=’*.git-backup*’ lg.zip ./

Remove Script Tags From HTML Files

Oct 29th, 2016 12:06 pm

The below script will remove all script tags from HTML files and creates a <FILE_NAME.html>.bak file (where <FILE_NAME.html> is actual file name) which have the actual content while the actual file is replaced with content removing script tag.

perl –i.bak –0777ne ‘s|<script.?||gms;print’ .html

Disable backup (.bak files):

perl -i -0777ne ‘s|<script.*?||gms;print’ *.html

Ignore file creation, just print the result:

perl -0777ne ‘s|<script.*?||gms;print’ *.html

Git Replace Local Version With Remote Version

Oct 29th, 2016 12:04 pm

git pull is the primary command while acquiring the remote changes, under the hood this is same as running git fetch and git merge but this won’t help when you are sure to replace the local content with remote while conflicts.

Using the below set of commands to hard reset local version with remote:

git fetch origin

git reset –hard origin/master

Explanation:

  • git fetch – This will imports commits from a remote repository into your local repo.
  • git reset –hard origin/master – This will reset Reset current HEAD to the specified state, Hard is to ignore any changes in your local copy.

Tip:

Make a alias git-replace: **alias git-replace=”git fetch origin && git reset –hard origin/master”**

Read more about: git reset, git pull, git fetch
