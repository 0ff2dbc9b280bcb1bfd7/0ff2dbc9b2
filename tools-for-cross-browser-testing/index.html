Tools for Cross-Browser Testing

Nov 6th, 2016 5:27 pm

With growing web, the browser in which where people view the webpage is for toss, for better user experience cross browser testing is hygiene for a website. Here are a few tools for testing a website for cross browser compatibility:

1. Locally Installed Browsers:

Have multiple browser installations in your local machine, here are the essentials browsers based on the market share:

  1. Google Chrome
  2. Firefox
  3. Safari
  4. IE
  5. Opera

If you are not on Windows you can download IE vm directly from Microsoft

2. Chrome’s mobile device emulator:

Chrome supports device emulation, this can be enabled by clicking mobile icon on Chrome’s developer tools, or using Ctrl + Shift + M after opening developer tools, here you can choose multiple devices or add a new device.

  1. Use browserstack.com for live browser testing (PAID)

  2. Use browsershots.org for screenshots of your web page in different operating systems and browsers. (FREE)
