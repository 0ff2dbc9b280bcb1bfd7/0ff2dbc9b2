Copy File From Remote Machine to Local Using SCP in Linux

Nov 6th, 2016 5:38 pm

Scp stands for secure copy program which copies files between hosts on a network. It uses ssh for data transfer, and uses the same authentication and provides the same security as ssh.

copy remote file to local:

scp -r joe@127.20.20.20:/home/joe/file.csv ~/remote/joe/

This will copy remote’s file.csv to the local directory ~/remote/joe
