Enable Date in Unity Taskbar in Ubuntu

Dec 10th, 2016 12:25 pm

Step 1 – Install dconf-editor:

sudo apt-get -y install dconf-editor

Step 2 – Change settings in dconf editor:

  1. Navigate through: com -> canonical -> indicator -> datetime

  2. Enable show-date

Tip – You can also enable show-day, show-year to show day and year in the task bar.
