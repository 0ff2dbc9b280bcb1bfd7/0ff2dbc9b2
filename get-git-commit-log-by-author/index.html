Get Git Commit Log by Author

Oct 29th, 2016 9:58 am

Git log’s –author flag can be used to list the commits made by a author, working example:

git log –author=”joe@gmail.com”

Git search for author is however not a strict comparision, if the given input exists in author, git prints that out, for example using @example.com in search field will list out all commits of author who uses example.com email:

git log –author=”@example.com”

Tips:

  • Using –no-pager flag prints output to stdout instead of pager (new window)
  • Using -1 will restrict git log to one result. (1 can be replaced with any numeric)

To get the recent commit made by a author use:

git log -1 –author=”joe@gmail.com”
