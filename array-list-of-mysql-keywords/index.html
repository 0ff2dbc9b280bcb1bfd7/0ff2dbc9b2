Array List of MySQL Keywords

Nov 15th, 2016 2:53 pm

All 612 MySQL keywords and reserved words have been listed at dev.mysql.com/doc/refman/5.5/en/keywords.html, however I you would like to use those in programming languages it should be converted into an array, I’ve written a simple script to convert the table into array list.

var elements = $("code.literal");
var keywords = [];
for(var i=0;i<elements.length;i++){
    keywords.push(elements[i].innerHTML);
}
console.log(JSON.stringify(keywords));
