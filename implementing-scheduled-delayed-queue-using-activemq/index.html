Implementing Scheduled / Delayed Queue Using ActiveMQ

Oct 29th, 2016 10:37 am

ActiveMQ has in-built support for scheduler, this adds support to deliver a message to the listener after x time. To enable this support add chedulerSupport=”true” in broker tag in activemq.xml, broker tag with scheduler support looks like this:

<broker xmlns=“http://activemq.apache.org/schema/core&#8221; brokerName=“localhost” dataDirectory=“${activemq.data}” schedulerSupport=“true”>

To schedule a ActiveMQ message any scheduled property need to be set using setLongProperty). Avaliable properties are:

  • AMQ_SCHEDULED_DELAY
  • AMQ_SCHEDULED_PERIOD
  • AMQ_SCHEDULED_REPEAT
  • AMQ_SCHEDULED_CRON
